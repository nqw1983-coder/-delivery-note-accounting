# iPad 使用说明(给小白用户)

## 第一次使用 — 加到桌面(管理员帮你做一次)

1. 在 iPad **Safari** 打开你的网址(管理员告诉你)
2. 输入访问密码
3. 点屏幕底部的 **分享按钮**(一个方框+向上的箭头)
4. 在弹出菜单中向下滑,找到 **"添加到主屏幕"**
5. 点 **"添加"**
6. 现在 iPad 桌面会出现 **"送货单"** 图标

**以后每次使用,点桌面这个图标就行**,不用再打开 Safari。

---

## 日常使用

### 录入一张送货单

1. 点桌面 **"送货单"** 图标打开
2. 左下角 **选择图片** → 拍照或从相册选
3. 点 **开始扫描**
4. 等几秒钟,识别完会自动弹出确认窗口
5. 看一下店家、日期、金额是否正确
   - 不对就直接改
6. 点 **保存**
7. 数据自动同步,Mac 端也能看到

### 一次拍多张

1. 选图片时可以多选(最多 10 张)
2. App 会一张一张让你确认
3. 自动学习:如果改了店家名,下次类似的字也会识别准

---

## 数据备份(建议每月做一次)

数据会自动同步到云端,但**建议你每月手动备份一次**,更安全:

1. 点左上角 **下载图标**(向下箭头)
2. 选 **"导出完整备份 (JSON)"**
3. 系统弹出 **"保存到..."**
4. 选 **iCloud Drive** 或 **分享到微信**(发给自己)

万一某天数据出问题,把这个文件给管理员就能恢复。

## 常见问题

### 网慢的时候

- 仍然可以录入,会先存在本地
- 等到 WiFi 连上,会自动上传到云端
- 看到顶部"已同步"就是成功了

### 不小心删了桌面图标

- 重新打开 Safari → 打开网址 → 重新加到主屏幕
- 或者找管理员帮你

### 拍的照片很模糊识别不出来

- 重新拍一张,光线足、单据平铺、字清楚
- 实在识别不出,可以直接在表格里手动输入金额

### 桌面图标点开后白屏

- 长按图标 → 删除图标
- 重新加到主屏幕(参考开头步骤)
- 一般是程序刚更新,清一下就好

---

## 🚨 三个"千万别"

1. **千万别删除桌面上的"送货单"图标** — 删了要重新设置
2. **千万别清理 Safari 浏览数据** — 可能影响 App 缓存
3. **千万别删除 iCloud 上的备份文件** — 数据保险
